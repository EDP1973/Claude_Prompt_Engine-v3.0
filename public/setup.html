🤖 OpenAI API Key Required for Chat

Powers iAI's conversation intelligence. Requires a paid OpenAI account with billing credit.

🔗 Open platform.openai.com/api-keys
  1. Sign in at platform.openai.com (create free account if needed)
  2. Click "+ Create new secret key"
  3. Name it: claude-prompt-engine-iai
  4. Select all permissions and click Create secret key
  5. Copy the key (it starts with sk-proj-)
  6. If quota errors: add billing at Billing Overview
Your key is stored only in .env on this server — never transmitted or logged.

🐙 GitHub Token Recommended

Enables GitHub API access and GitHub Copilot as fallback AI engine.

🔗 Open github.com/settings/tokens/new
  1. Sign in to github.com
  2. The link pre-fills name + scopes — click Generate token
  3. Required scopes: repo, read:user, read:org
  4. For GitHub Copilot: also enable copilot scope
  5. Copy the token (starts with ghp_)

📞 Telnyx API Key Optional

Required only for VoIP/SIP telephony, Vicidial dialplan generation, and call routing.

🔗 Open portal.telnyx.com
  1. Sign in at portal.telnyx.com
  2. Go to Auth → API Keys
  3. Click Create API Key
  4. Copy the key (starts with KEY_)

🗄️ MySQL / Vicidial Database Optional

Connection settings for your Vicidial MySQL database. Leave blank to use SQLite-only mode.
