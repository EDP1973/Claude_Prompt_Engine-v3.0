## 7️⃣ API Reference

### Generate Prompt
```bash
POST /api/generate
Content-Type: application/json

{
  "type": "coding",
  "language": "JavaScript",
  "task": "Create a function to fetch data",
  "constraints": ["error handling", "TypeScript"],
  "level": "intermediate",
  "model": "Claude",
  "purpose": "web-app"
}

Response: {
  "success": true,
  "prompt": "Complete generated prompt..."
}
```

### Get Models
```bash
GET /api/models
Response: {
  "models": [
    {"id": "claude", "name": "Claude", ...},
    ...
  ]
}
```

### Get Carriers
```bash
GET /api/carriers
Response: {
  "carriers": [
    {
      "id": "twilio",
      "name": "Twilio",
      "features": [...],
      "pricing": "Pay-per-use",
      ...
    },
    ...
  ]
}
```

### All Endpoints
- `POST /api/generate` - Generate prompt
- `GET /api/models` - List LLMs
- `GET /api/purposes` - List purposes
- `GET /api/templates` - List templates
- `GET /api/carriers` - List carriers
- `GET /api/dialplans` - List dial plans
- `GET /api/ai-config` - List AI models
- `GET /api/extensions` - List extensions
- `GET /api/history` - Get history
- `GET /api/stats` - Get statistics
- `POST /api/memory/clear` - Clear history

## 🚀 Getting Started

1. **Install**
   ```bash
   npm install
   ```

2. **Start Server**
   ```bash
   npm start
   ```

3. **Open Browser**
   - Navigate to http://localhost:3000

4. **Generate Your First Prompt**
   - Select Claude model
   - Choose JavaScript language
   - Pick "Web App" purpose
   - Select "Coding" type
   - Enter your task
   - Click "Generate"

5. **Explore Advanced Features**
   - Check Telephony carriers
   - Review Dial Plans
   - Configure AI settings
   - Browse Extensions
   - Read Personal AI Guide
