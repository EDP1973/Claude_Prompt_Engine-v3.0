🚀 How to Use Prompt Engine

1. Select Your Model

Choose the AI language model you'll be using (Claude, GPT-4, etc.). This optimizes the prompt for that specific model's strengths.

2. Pick Your Language & Level

Select your programming language and skill level. This ensures the generated code matches your expertise and requirements.

3. Define Your Purpose

Specify what you're building - mobile app, API, CLI tool, etc. This shapes the entire approach and architecture suggestions.

4. Choose Prompt Type

Coding: Generate new code | Debugging: Fix issues | Refactor: Improve quality | Testing: Write tests | Documentation: Create docs | Architecture: Design systems | Code Review: Analyze code | Optimization: Boost performance

5. Write Your Task

Be specific! The more detail you provide, the better the generated prompt. Include requirements, edge cases, and preferences.

6. Add Constraints (Optional)

Specify any additional requirements like "must use TypeScript", "include error handling", "responsive design", etc.
