🔧 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────────

"Can't connect to localhost:3000"
  → Check if service is running (see command line section)
  → Check if port 3000 is available (not in use)
  → Try different port: set PORT=3001 & start.bat (Windows)

"npm install failed during installation"
  → Requires internet connection to download dependencies
  → Check your internet connection
  → Try reinstall

"Service won't start"
  → Check logs (see command line section)
  → Try manual start: node server.js
  → Check disk space (need 500 MB)

"Permission denied errors"
  → May need admin/sudo rights
  → Check file permissions in installation directory
